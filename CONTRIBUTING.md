# Contributing

**Bug reports and questions are welcome** — please open an [issue](../../issues).

This project is **not accepting external code contributions** at this time:
pull requests generally won't be reviewed or merged, and `main` is protected.
If you want to change the code, please **fork** and build your own version
under the terms of the [MIT License](LICENSE).

For security vulnerabilities, do **not** open a public issue — see
[SECURITY.md](SECURITY.md).
